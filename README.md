# Library Management Website

A full-stack web-based library management system built using Flask and Vue.js.  
This application allows users to browse books, make reservations, and place orders, while librarians can manage library data and publish articles.

---

## Tech Stack

### Frontend
- Vue.js
- HTML, CSS, JavaScript
- Deployed on **Netlify**

### Backend
- Python (Flask)
- REST API
- Deployed on **DewaCloud**

### Database
- MySQL / SQLite
- SQLAlchemy ORM

---

## Features

### User
- View book catalog
- Search books
- Reserve books
- Place book orders
- View library articles/news

### Librarian (Admin)
- Manage book data (CRUD)
- Manage reservations and orders
- Publish library news/articles
